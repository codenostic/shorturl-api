URL Shortener Service

User stories:
  1. Pass a valid URL as a 'query' and you get back a short url
  2. Pass a valid short url and you will be redirected to original url

Example usage:

1) https://fcc-header-ms.glitch.me/url/?query=http://www.google.com/
2) https://fcc-header-ms.glitch.me/12435

Example output:

1) https://fcc-header-ms.glitch.me/12435
2) Redirected to http://google.com
